# 从门电路到 sCPU：Logisim 位切片 CPU 设计实战

## 📖 项目简介
本项目是基于 Logisim 仿真软件，**从零开始手工搭建**的一款支持 `li`、`add` 和 `bner` 指令的 4 位简单 CPU（sCPU）。
不同于直接使用 Logisim 自带的寄存器堆元件，本项目的核心亮点是：**完全采用“位切片（Bit-sliced）”架构**，将 1 位的 A0 模块垂直切分、并行组合，从而拼装出 4 位乃至 8 位的寄存器堆。

## 🎯 已实现的功能特性
- **底层门电路级构建**：使用基本逻辑门、D触发器、多路选择器、译码器等，不依赖高级抽象组件。
- **取指与译码**：2位 / 3位 PC 程序计数器，自动递增；8位 ROM 实现指令存储；分线器拆分指令字段。
- **指令支持**：
  - `li rd, imm`：加载立即数到目标寄存器。
  - `add rd, rs1, rs2`：寄存器加法，支持双端口读取。
  - `bner rs1, rs2, addr`：比较两寄存器，不相等时跳转（跳转地址最低2位有效）。
- **循环与输出**：实现了 8 条指令循环计算 `1+3+5+7+9`，最终计算得出十进制 `25`（十六进制 `19`），并通过七段数码管稳定输出。
- **硬件调试能力**：成功排查并解决了包括 MUX 悬空、分线器位映射错位、写使能（WE）错接到复位端（R）等多个实际电路 Bug。

## 🛠️ 技术栈
- **仿真工具**：Logisim
- **体系结构**：位切片架构（Bit-sliced）、8位 RISC 风格指令集
- **硬件理论**：组合逻辑、时序逻辑、状态机（FSM）
